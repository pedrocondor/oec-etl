# Canadian Sub-National Trade Data

Canadian International Merchandise Trade Database (CIMT) provide sub-national import/export data.  

Product resolution : 6 digit HS classification  

Spatial resoultion : Province  

Frequency : Monthly

Starting Year : 1992
